Global Weather Study

For this study, I was tasked with building weather dataset of over 500 cities around the globe. Python allowed me to build a list of random geo-coordinates to use with the citipy package which takes those coordinates and locates the nearest city around the globe. With the list of city names citipy helped me create, I began using the Open Weather Map API to request the current data. With this data I built a Pandas dataframe to organize and analyze the data.

Equipped with this fresh data for 2200+ cities, I analyzed the data and noticed some patterns. We specifically analyzed cloudiness, humidity, temperature and wind speed. These patterns easily became visible when visualized with matplotlib. The visualizations all share the same x-axis units, Latitude, this allows for easy comparison and and understanding.
